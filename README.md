# Exploratory Data Analysis (EDA) Notebook

This repository contains a complete Exploratory Data Analysis (EDA) notebook created to understand the structure, quality, and patterns within a dataset before applying machine learning models.  
It includes data cleaning, descriptive statistics, visualization techniques, correlation analysis, and initial feature engineering.

---

##  Notebook Included
- **Exploratory_Data_Analysis(EDA).ipynb**  
  A comprehensive EDA workflow demonstrating:
  - Dataset loading & inspection  
  - Handling missing values  
  - Outlier detection  
  - Summary statistics  
  - Data distributions & histograms  
  - Boxplots, pairplots, and scatter plots  
  - Correlation matrix and heatmaps  
  - Feature relationships and insights  

---

##  Objective
This notebook is designed for:
- Strengthening core data analysis skills  
- Understanding dataset structure before modeling  
- Identifying patterns and issues early in the ML pipeline  
- Practicing visualization and interpretation
